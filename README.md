# Holographic Interactive AI

This project is a Holographic interactive AI system that brings together voice, animation, and physical hardware. The goal is to create a holographic character that can listen, talk, respond, and move in a more lifelike way in Air looks like real human standing.

## Project Aim

The main aim of this project is to build an interactive AI assistant that feels more real and engaging. Instead of only being a voice bot, this system combines visual animation, smart conversation, and hardware control to create a complete experience.

## Main Objectives

- Create a character that can appear and behave like a live assistant.
- Connect a Unity-based visual interface with a Python AI backend.
- Enable voice interaction so the system can listen and respond.
- Use hardware components to make the project feel more physical and interactive.
- Record and organize media for demonstrations, testing, and presentation.

## What Things This Project includes

- A Unity client has been set up for the character interface and animation.
- A Python backend handles the AI logic and communication flow.
- Hardware firmware has been prepared for control of motors, sensors, and related components.
- Media folders have been created to store final results, hardware preparation steps, and software preparation steps.
- The project is now organized into clear parts for development, testing, and presentation.

## Project Structure

- Unity-Client: Contains the Unity project, character visuals, animations, and interactive interface.
- Python-Backend: Contains the AI logic, voice processing, and communication code.
- Hardware-Firmware: Contains the Arduino, ESP32, and Teensy code for hardware control.
- Media: Contains final videos and images, along with preparation materials for hardware and software.
- Blender_Char_Animations: Contains Blender animation files and character animation assets.

## Technology Used

- Frontend: Unity with C#
- Backend: Python
- Hardware: Arduino, ESP32, Teensy, motors, and sensors
- Communication: Serial and web-based communication methods

## Overall Summary

This project is a step toward building a smart, interactive holographic assistant that combines software intelligence with visual animation and physical hardware. It is still being developed and improved, but the foundation for a complete interactive experience is already in place.

## How to Run the Project

1. Open the Unity project from the Unity-Client folder.
2. Make sure the required Unity packages are installed.
3. Open the Python-Backend folder in VS Code.
4. Install Python packages from requirements.txt. (first create virtual environmnet )
5. Run the Python backend script to start the AI service.
6. Open the Unity scene and start the project.
7. Connect the hardware components and upload the firmware from the Hardware-Firmware folder.
8. Test the full system and adjust settings if needed.
