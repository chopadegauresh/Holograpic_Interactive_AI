# Unity Client

This folder contains the Unity project for the interactive character.

## Folders and what they do

- Assets: Main project files like scenes, scripts, models, sounds, and materials.
- Packages: Extra Unity packages used by the project.
- ProjectSettings: Unity project settings such as input, graphics, and build settings.
- Library: Unity’s working files for the project.
- Logs: Error and debug logs from Unity.
- obj: Temporary build files.
- Temp: Temporary files created while Unity runs.
- UserSettings: Personal settings for this Unity project.
